Purebzar Live Preview

*Note: PureBzar is an Arabic business based in the Arab Emirates, which made it a requirement that this website was built with RTL support in mind and the Arabic Language

Overview

"PureBzar" is both the name of the company I created an ecommerce website for using the Shopify platform, and the name of the custom-built theme the website was built (by me) on.

The theme included full integration with Shopify's "storefront API" (cart, checkout, etc), as well as a few customization options that made it easy to modify the look of the website without editing code.

It included slideshows and product slide displays which you usually get through Shopify plugins.
The website's design was custom. But images and art designed specifically for the company's brand and product were given to me by the designers of Wateen Plus

The final theme contained a compact 4 .liquid templates, covering most of the website's pages, and a dozen custom-built Shopify "Sections"

The theme was fully responsive and worked just as well on mobile

All in all, the theme had everything you'd expect from the common ecommerce website (cart page before checkout, individual product page, social media links, etc)
